Propiedades CSS según su finalidad

  • Propiedades de diseño visual: Estas propiedades se utilizan para controlar la apariencia visual de los elementos en una página web.

    Algunas de estas propiedades incluyen:
    color, background-color, border, margin, padding, width, height, font-size, text-align, display, position, z-index.

  • Propiedades de animación y transición: Estas propiedades se utilizan para crear animaciones y efectos de transición en los elementos de la página web.

    Algunas de estas propiedades incluyen:
    animation, transition, transform, opacity.

  • Propiedades de diseño responsivo: Estas propiedades se utilizan para hacer que una página web sea adaptable a diferentes tamaños de pantalla.

    Algunas de estas propiedades incluyen:
    media queries, viewport, flexbox, grid.

  • Propiedades de texto: Estas propiedades se utilizan para controlar la apariencia del texto en una página web.

    Algunas de estas propiedades incluyen:
    font-family, line-height, letter-spacing, text-decoration, text-shadow.

  • Propiedades de diseño de página: Estas propiedades se utilizan para controlar la distribución y el diseño de los elementos en una página web.

    Algunas de estas propiedades incluyen:
    float, clear, column-count, column-gap, column-rule.
