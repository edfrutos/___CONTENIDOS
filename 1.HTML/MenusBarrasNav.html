Ejemplo básico de código HTML y CSS para crear un menú y una barra de navegación:

  • Home
  • About
  • Contact

Este ejemplo crea una barra de navegación con un fondo de color gris oscuro, y una lista de enlaces apilados horizontalmente. Los enlaces tienen un fondo gris más claro al pasar el cursor sobre ellos. Es importante mencionar que este ejemplo es básico y puede variar dependiendo de tus necesidades y preferencias.
