Este texto se desplaza Este es otro texto que se desplaza ¡Y este también se mueve!

En este caso, utilizamos una estructura HTML diferente, en la que el texto que se desplaza se coloca dentro de un contenedor de clase "marquee__inner" y se repite tres veces con tres elementos "span". Luego, animamos el contenedor de "marquee__inner" utilizando una transformación "translate" que mueve el elemento desde la posición inicial en el 100% de la posición izquierda del contenedor al -100% de la posición izquierda. La propiedad "white-space: nowrap" evita que el texto se envuelva al siguiente renglón.
