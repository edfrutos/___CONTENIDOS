Propiedades CSS de diseño texto

  • color: Establece el color del texto.
  • direction: Define la dirección del texto.
  • font: Establece varias propiedades de la fuente en un solo valor.
  • font-family: Establece la familia de fuentes que se utilizará para el texto.
  • font-size: Establece el tamaño de la fuente del texto.
  • font-style: Define el estilo de la fuente.
  • font-weight: Define el grosor de la fuente.
  • letter-spacing: Establece el espaciado entre caracteres.
  • line-height: Define la altura de línea del texto.
  • list-style: Establece varias propiedades de estilo de lista en un solo valor.
  • list-style-type: Define el tipo de marcador de lista.
  • text-align: Establece la alineación del texto.
  • text-decoration: Define la decoración del texto.
  • text-indent: Establece la sangría del primer renglón del texto.
  • text-transform: Define la transformación de texto.
  • vertical-align: Define la alineación vertical del texto.
  • white-space: Define cómo se manejan los espacios en blanco en el texto.
  • word-spacing: Establece el espaciado entre palabras.
  • text-shadow: Crea una sombra detrás del texto.
  • text-overflow: Define cómo se maneja el desbordamiento de texto en un elemento.
  • text-rendering: Define la calidad de renderizado de texto en la pantalla.
  • text-size-adjust: Permite ajustar el tamaño de la fuente en dispositivos móviles.
  • word-break: Define cómo se deben dividir las palabras en el texto cuando no hay espacio suficiente para mostrar una línea completa.
  • word-wrap: Define cómo se deben envolver las palabras largas en el texto cuando no hay suficiente espacio para mostrar una línea completa.
  • hyphens: Controla la división de palabras cuando el texto se desborda de un contenedor.
  • overflow-wrap: Controla el comportamiento de desbordamiento de palabras largas en una línea.
  • tab-size: Define el tamaño de tabulación.
  • text-align-last: Define cómo se alinea el último renglón del texto en un bloque que tiene más de una línea.
  • text-justify: Define cómo se justifica el texto dentro de un bloque que tiene más de una línea.
  • text-outline: Establece un contorno alrededor del texto.
  • text-underline-position: Controla la posición del subrayado del texto.
  • font-display: Controla cómo se muestra una fuente mientras se está descargando.
  • font-feature-settings: Permite activar y desactivar características tipográficas en las fuentes.
  • font-kerning: Controla el espacio entre caracteres en una fuente.
  • font-variant: Controla las variantes de una fuente, como versalitas o números antiguos.
  • font-variant-caps: Controla las variantes de capitalización de una fuente, como versalitas o small caps.
  • font-variant-ligatures: Controla si se usan o no ligaduras en una fuente.
  • font-variant-numeric: Controla la forma en que se muestran los números en una fuente, como números antiguos o fraccionarios.
  • line-break: Controla cómo se deben romper las líneas de texto.
  • text-combine-upright: Controla la forma en que se muestran varios caracteres en una línea vertical.
  • text-orientation: Controla la orientación del texto en un bloque que tiene más de una línea.
  • text-underline-offset: Controla la distancia entre el subrayado y el texto.
  • unicode-bidi: Controla la dirección de los caracteres en un elemento.
  • word-spacing: Establece la distancia entre palabras.
  • hanging-punctuation: Controla la forma en que se muestran los signos de puntuación al final de una línea.
  • font-stretch: Controla el ancho de una fuente, como condensado o extendido.
  • text-emphasis: Controla el énfasis en el texto, como resaltado o puntas de flecha.
  • text-emphasis-color: Establece el color del énfasis en el texto.
  • text-emphasis-style: Define el estilo de énfasis en el texto.
  • text-emphasis-position: Controla la posición del énfasis en el texto.
  • text-underline: Define si se debe subrayar el texto y establece su estilo y color.
  • text-justify-trim: Controla el recorte del espacio en los elementos justificados.
  • text-align-self: Alinea el texto dentro de un contenedor flex, incluso si el texto no está en un elemento de bloque.
  • text-orientation: Controla la orientación del texto en un bloque que tiene más de una línea.
