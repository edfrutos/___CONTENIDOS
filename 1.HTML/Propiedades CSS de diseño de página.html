Propiedades CSS de diseño de página

  • align-content: Esta propiedad se utiliza para alinear el contenido dentro de un contenedor de línea o de bloque a lo largo del eje de la cuadrícula secundaria.
  • align-items: Esta propiedad se utiliza para alinear los elementos secundarios de un contenedor en el eje principal.
  • align-self: Esta propiedad se utiliza para alinear un elemento secundario dentro de su contenedor en el eje principal.
  • display: Esta propiedad se utiliza para definir el tipo de visualización de un elemento, como bloque, en línea o en línea-bloque.
  • flex: Esta propiedad se utiliza para especificar cómo se deben distribuir los elementos secundarios dentro de un contenedor flexible.
  • flex-basis: Esta propiedad se utiliza para establecer la base flexible de un elemento secundario en un contenedor flexible.
  • flex-direction: Esta propiedad se utiliza para establecer la dirección en la que los elementos secundarios de un contenedor flexible se distribuyen.
  • flex-flow: Esta propiedad es una abreviatura de las propiedades flex-direction y flex-wrap, y se utiliza para establecer la dirección y el ajuste de los elementos secundarios en un contenedor flexible.
  • flex-grow: Esta propiedad se utiliza para especificar la cantidad de espacio adicional que un elemento secundario puede ocupar en un contenedor flexible.
  • flex-shrink: Esta propiedad se utiliza para especificar la cantidad de espacio que un elemento secundario puede encogerse en un contenedor flexible.
  • flex-wrap: Esta propiedad se utiliza para especificar si los elementos secundarios de un contenedor flexible deben ajustarse en una sola línea o en varias líneas.
  • float: Esta propiedad se utiliza para especificar cómo un elemento debe flotar en relación con el contenido de la página.
  • grid: Esta propiedad se utiliza para crear una cuadrícula flexible de filas y columnas para organizar el contenido de una página.
  • grid-area: Esta propiedad se utiliza para especificar la ubicación de un elemento dentro de una cuadrícula de diseño de página.
  • grid-auto-rows: Esta propiedad se utiliza para especificar la altura predeterminada de las filas automáticas de una cuadrícula de diseño de página.
  • grid-column: Esta propiedad se utiliza para especificar la ubicación de un elemento en una cuadrícula de diseño de página en función de las columnas de la cuadrícula.
  • grid-row: Esta propiedad se utiliza para especificar la ubicación de un elemento en una cuadrícula de diseño de página en función de las filas de la cuadrícula.
  • grid-template-areas: Esta propiedad se utiliza para definir una cuadrícula de diseño de página utilizando áreas de diseño con nombres.
  • grid-template-columns: Esta propiedad se utiliza para especificar las columnas de una cuadrícula de diseño de página.
  • grid-template-rows: Esta propiedad se utiliza para especificar las filas de una cuadrícula de diseño de página.
  • justify-content: Esta propiedad se utiliza para alinear los elementos secundarios de un contenedor a lo largo del eje principal.
  • margin: Esta propiedad se utiliza para definir el espacio exterior de un elemento.
  • max-height: Esta propiedad se utiliza para definir la altura máxima de un elemento.
  • max-width: Esta propiedad se utiliza para definir el ancho máximo de un elemento.
  • min-height: Esta propiedad se utiliza para definir la altura mínima de un elemento
  • min-width: Esta propiedad se utiliza para definir el ancho mínimo de un elemento.
  • order: Esta propiedad se utiliza para especificar el orden de un elemento secundario en un contenedor flexible.
  • overflow: Esta propiedad se utiliza para especificar cómo se debe gestionar el contenido que desborda el área de un elemento.
  • padding: Esta propiedad se utiliza para definir el espacio interior de un elemento.
  • position: Esta propiedad se utiliza para establecer la posición de un elemento en relación con su contenedor.
  • top: Esta propiedad se utiliza para establecer la distancia desde la parte superior de un elemento hasta el borde superior del contenedor.
  • right: Esta propiedad se utiliza para establecer la distancia desde el borde derecho de un elemento hasta el borde derecho del contenedor.
  • bottom: Esta propiedad se utiliza para establecer la distancia desde la parte inferior de un elemento hasta el borde inferior del contenedor.
  • left: Esta propiedad se utiliza para establecer la distancia desde el borde izquierdo de un elemento hasta el borde izquierdo del contenedor.
  • transform: Esta propiedad se utiliza para aplicar transformaciones en 2D o 3D a un elemento.
  • transition: Esta propiedad se utiliza para definir una transición animada entre diferentes estados de un elemento.
  • width: Esta propiedad se utiliza para definir el ancho de un elemento.
  • background: Esta propiedad se utiliza para definir el fondo de un elemento y puede incluir color, imagen o ambas.
  • border: Esta propiedad se utiliza para definir el borde de un elemento y puede incluir ancho, estilo y color.
  • box-shadow: Esta propiedad se utiliza para agregar una sombra alrededor de un elemento.
  • clear: Esta propiedad se utiliza para definir cómo se debe gestionar el contenido después de un elemento que ha sido flotado.
  • clip-path: Esta propiedad se utiliza para recortar una imagen o forma para que se ajuste a una forma determinada.
  • column-count: Esta propiedad se utiliza para especificar el número de columnas en un diseño de múltiples columnas.
  • column-gap: Esta propiedad se utiliza para definir el espacio entre columnas en un diseño de múltiples columnas.
  • column-rule: Esta propiedad se utiliza para definir la regla que separa las columnas en un diseño de múltiples columnas.
  • filter: Esta propiedad se utiliza para aplicar efectos visuales a un elemento, como desenfoque, saturación, contraste, brillo, entre otros.
  • font: Esta propiedad se utiliza para definir el tipo de letra, el tamaño, el peso y el estilo del texto.
  • line-height: Esta propiedad se utiliza para definir la altura de línea del texto.
  • opacity: Esta propiedad se utiliza para definir la opacidad de un elemento, lo que puede hacer que sea más o menos transparente.
  • text-align: Esta propiedad se utiliza para alinear el texto dentro de un elemento, como a la izquierda, a la derecha o al centro.
  • text-decoration: Esta propiedad se utiliza para agregar decoración de texto, como subrayado, tachado o resaltado.
  • text-transform: Esta propiedad se utiliza para cambiar la capitalización de un texto, como a mayúsculas, a minúsculas o a la capitalización inicial.
  • vertical-align: Esta propiedad se utiliza para alinear verticalmente un elemento en relación con el contenido de la página.
  • white-space: Esta propiedad se utiliza para definir cómo se deben gestionar los espacios en blanco dentro de un elemento, como colapsarlos o conservarlos.
