Propiedades CSS de diseño responsivo

  • align-content: alinea el contenido verticalmente en un contenedor con varias líneas.
  • align-items: alinea los elementos verticalmente dentro de un contenedor.
  • align-self: alinea un elemento en particular dentro de un contenedor.
  • display: especifica cómo se mostrará un elemento, como bloque, en línea, en línea-block, etc.
  • flex: establece la base para el diseño flexible de elementos dentro de un contenedor.
  • flex-basis: establece la base para la dimensión de un elemento flexible.
  • flex-direction: establece la dirección del flujo de elementos dentro de un contenedor flexible.
  • flex-flow: es una propiedad abreviada que combina flex-direction y flex-wrap.
  • flex-grow: especifica cómo se distribuye el espacio adicional en un contenedor flexible.
  • flex-shrink: especifica cómo se reduce el espacio en un contenedor flexible.
  • flex-wrap: especifica si los elementos deben envolverse o no cuando no hay suficiente espacio en un contenedor flexible.
  • justify-content: alinea los elementos horizontalmente dentro de un contenedor.
  • margin: establece el margen de un elemento.
  • max-width: establece la anchura máxima de un elemento.
  • min-width: establece la anchura mínima de un elemento.
  • overflow: especifica cómo manejar el desbordamiento del contenido dentro de un elemento.
  • padding: establece el relleno de un elemento.
  • width: establece la anchura de un elemento.
  • aspect-ratio: establece la relación de aspecto de un elemento.
  • bottom: establece la posición del borde inferior de un elemento.
  • box-sizing: especifica cómo se calcula el ancho y alto de un elemento, incluyendo su relleno y borde.
  • clear: especifica si un elemento debe despejar los elementos flotantes a su alrededor.
  • float: establece si un elemento debe flotar a la izquierda, a la derecha o no flotar en absoluto.
  • font-size: establece el tamaño de fuente de un elemento.
  • font-weight: establece el grosor de la fuente de un elemento.
  • height: establece la altura de un elemento.
  • left: establece la posición del borde izquierdo de un elemento.
  • line-height: establece la altura de línea de un elemento.
  • max-height: establece la altura máxima de un elemento.
  • min-height: establece la altura mínima de un elemento.
  • opacity: establece la opacidad de un elemento.
  • position: establece la posición de un elemento.
  • right: establece la posición del borde derecho de un elemento.
  • top: establece la posición del borde superior de un elemento.
  • transition: permite una transición suave entre los estados de un elemento.
  • vertical-align: establece la alineación vertical de un elemento dentro de su línea base.
  • visibility: especifica si un elemento debe ser visible o no.
  • z-index: establece el orden de apilamiento de un elemento en relación con otros elementos.
  • background: establece el fondo de un elemento, incluyendo color, imagen y posición.
  • background-size: establece el tamaño de la imagen de fondo de un elemento.
  • border: establece el ancho, estilo y color del borde de un elemento.
  • border-radius: establece el radio de los bordes de un elemento.
  • box-shadow: establece la sombra del cuadro de un elemento.
  • clip: especifica un área de recorte para un elemento posicionado.
  • cursor: establece el cursor del ratón sobre un elemento.
  • filter: aplica efectos visuales a un elemento, como desenfoque, brillo y saturación.
  • gradient: permite crear gradientes de color en un elemento.
  • grid: establece el diseño de cuadrícula de un elemento y sus elementos secundarios.
  • letter-spacing: establece el espacio adicional entre caracteres de un elemento.
  • outline: establece un contorno alrededor de un elemento sin afectar su posición.
  • overflow-x: especifica cómo manejar el desbordamiento horizontal del contenido dentro de un elemento.
  • overflow-y: especifica cómo manejar el desbordamiento vertical del contenido dentro de un elemento.
  • perspective: establece la perspectiva 3D de un elemento.
  • perspective-origin: establece el punto de vista 3D de un elemento.
  • resize: establece si un elemento es redimensionable por el usuario.
  • text-align: establece la alineación del texto en un elemento.
  • text-decoration: establece la decoración del texto, como subrayado y tachado.
  • transform: permite transformar la forma, tamaño y posición de un elemento.
  • transform-origin: establece el punto de transformación de un elemento.
  • user-select: establece si un usuario puede seleccionar el texto dentro de un elemento.
  • animation: permite crear animaciones personalizadas para un elemento.
  • backface-visibility: establece si el reverso de un elemento visible durante una transformación 3D.
  • columns: establece el diseño en columnas de un elemento.
  • column-count: establece el número de columnas en un elemento.
  • column-gap: establece el espacio entre las columnas de un elemento.
  • column-rule: establece el ancho, estilo y color de la regla entre las columnas de un elemento.
  • counter-increment: incrementa el valor de un contador definido en un elemento.
  • counter-reset: reinicia el valor de un contador definido en un elemento.
  • hyphens: establece cómo se deben dividir las palabras en un elemento.
  • image-rendering: establece la calidad de renderizado de una imagen en un elemento.
  • object-fit: establece cómo una imagen o video debe ajustarse en un contenedor.
  • object-position: establece la posición de una imagen o video dentro de su contenedor.
  • opacity: establece la opacidad de un elemento.
  • page-break-after: establece si se debe iniciar una nueva página después de un elemento.
  • page-break-before: establece si se debe iniciar una nueva página antes de un elemento.
  • page-break-inside: establece si un elemento puede ser dividido en varias páginas durante la impresión.
  • pointer-events: establece si un elemento debe responder a eventos del mouse.
  • word-break: establece cómo se deben dividir las palabras largas en un elemento.
  • word-spacing: establece el espacio adicional entre palabras en un elemento.
  • writing-mode: establece la dirección de escritura del texto en un elemento.
