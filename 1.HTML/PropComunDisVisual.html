Propiedades CSS comunes de diseño visual

  • color: Define el color del texto.
  • font-family: Define la fuente utilizada para el texto.
  • font-size: Define el tamaño de la fuente del texto.
  • font-weight: Define el grosor de la fuente del texto.
  • text-align: Define la alineación del texto dentro del contenedor.
  • line-height: Define la altura de línea del texto dentro del contenedor.
  • background-color: Define el color de fondo del contenedor.
  • background-image: Define la imagen de fondo del contenedor.
  • background-size: Define el tamaño de la imagen de fondo del contenedor.
  • border: Define los bordes alrededor del contenedor, incluyendo color, grosor y estilo.
  • border-radius: Define el radio de los bordes del contenedor.
  • box-shadow: Define una sombra alrededor del contenedor.
  • margin: Define el espacio exterior alrededor del contenedor.
  • padding: Define el espacio interior entre el borde y el contenido del contenedor.
  • display: Define cómo se muestra un elemento, como bloque, en línea o en línea con el nivel de línea.
  • position: Define la posición de un elemento en relación con el contenedor principal.
  • top, right, bottom, left: Estas propiedades se usan con la propiedad position para definir la posición exacta de un elemento en la página.
  • float: Permite que un elemento flote a la izquierda o derecha de su contenedor.
  • clear: Define si un elemento flotante debe estar a la izquierda, a la derecha o no debe flotar en relación con otros elementos en la página.
  • z-index: Define la profundidad de un elemento en relación con otros elementos en la página.
  • text-decoration: Define la decoración del texto, como subrayado, tachado, etc.
  • text-transform: Define cómo se muestra el texto, como en mayúsculas, minúsculas, etc.
  • opacity: Define la transparencia de un elemento.
  • visibility: Define si un elemento es visible o no.
  • text-shadow: Define una sombra alrededor del texto.
  • box-sizing: Define si se debe incluir el relleno y el borde de un elemento en su ancho y altura total.
  • outline: Define un borde alrededor de un elemento que no afecta el tamaño o la posición del elemento.
  • outline-offset: Define la distancia entre el borde de un elemento y su contorno.
  • cursor: Define el tipo de cursor que se muestra cuando el mouse se coloca sobre un elemento.
  • user-select: Define si el texto dentro de un elemento se puede seleccionar o no.
  • resize: Define si un elemento se puede cambiar de tamaño y en qué direcciones.
  • transition: Define una transición suave entre los estados de un elemento.
  • transform: Define una transformación visual en un elemento, como rotación, escala, etc.
  • perspective: Define la perspectiva 3D para un elemento que se utiliza con transform.
  • flexbox: Define un sistema de diseño flexible que permite la distribución dinámica de elementos en un contenedor.
  • grid: Define una rejilla de diseño que permite una distribución más avanzada y personalizable de elementos en un contenedor.
  • align-items: Define la alineación vertical de los elementos en un contenedor.
  • justify-content: Define la alineación horizontal de los elementos en un contenedor.
  • flex-direction: Define la dirección en la que se colocan los elementos en un contenedor flexible.
  • grid-template-columns: Define las columnas en una rejilla de diseño.
  • grid-template-rows: Define las filas en una rejilla de diseño.
  • grid-column: Define la ubicación de un elemento en una rejilla de diseño en términos de columnas.
  • grid-row: Define la ubicación de un elemento en una rejilla de diseño en términos de filas.
  • grid-area: Define la ubicación de un elemento en una rejilla de diseño en términos de filas y columnas.
  • gap: Define el espacio entre elementos en una rejilla de diseño o contenedor flexible.
  • background-attachment: Define si la imagen de fondo del elemento se desplaza con el resto del contenido o se mantiene fija en su posición.
  • background-position: Define la posición de la imagen de fondo del elemento.
  • background-repeat: Define si la imagen de fondo del elemento se repite o no.
  • background-blur: Define el desenfoque aplicado a la imagen de fondo del elemento.
  • border-collapse: Define si los bordes de las celdas en una tabla se combinan o no.
  • caption-side: Define la posición de la leyenda de una tabla.
  • object-fit: Define cómo se ajusta el contenido dentro de un elemento contenedor.
  • object-position: Define la posición del contenido dentro de un elemento contenedor.
  • text-overflow: Define cómo se maneja el texto que desborda el ancho del contenedor.
  • white-space: Define cómo se maneja el espacio en blanco dentro de un elemento.
  • box-decoration-break: Define cómo se maneja la decoración de borde, como los bordes y los rellenos, alrededor de elementos divididos en varias líneas.
  • clip-path: Define una forma para recortar un elemento visualmente.
  • image-rendering: Define cómo se muestra una imagen, como suavizado o nitidez.
  • mix-blend-mode: Define cómo se mezcla el color de un elemento con el color del fondo o con otro elemento superpuesto.
  • text-orientation: Define la orientación del texto vertical u horizontal en un elemento.
  • text-combine-upright: Define cómo se combinan los caracteres verticales en el texto en un elemento.
  • text-justify: Define cómo se justifica el texto en un elemento.
  • word-break: Define cómo se rompen las palabras largas en líneas de texto en un elemento.
  • word-wrap: Define cómo se ajustan las palabras largas en un contenedor cuando no hay suficiente espacio para ellas.
  • writing-mode: Define la dirección de escritura del texto, como de izquierda a derecha o de arriba hacia abajo.
