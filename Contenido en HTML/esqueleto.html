Para comenzar con el diseño de una página web, puedes seguir estos estilos básicos:

Estos son estilos básicos y puedes personalizarlos según tus necesidades y preferencias.

  • Body: Fija el tamaño de fuente, color de fondo y margen de la página completa.

  • Header: Contiene el logotipo y la información de navegación. Puedes establecer un tamaño de fuente más grande y un color de fondo distinto para destacarlo.

  • Nav: Establece los estilos de navegación, como tamaño de fuente, alineación y color de fondo.

  • Main: Contiene el contenido principal de la página, incluyendo imágenes, texto y otros elementos. Establece un tamaño de fuente y color de fondo para este área.

  • Footer: Contiene información adicional como enlaces de pie de página, derechos de autor y contacto. Puedes establecer un tamaño de fuente más pequeño y un color de fondo distinto para diferenciarlo.

    • En este ejemplo, se aplican estilos a los elementos body, header, nav, main y footer mediante CSS. También se utiliza un div para separar los elementos dentro de la cabecera.
